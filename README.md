# Tic-Tac-Toe – Reinforcement Learning Agent spielen

## Aufgabe
In diesem Repository lernen Sie, wie ein **Reinforcement-Learning-Agent** Tic-Tac-Toe spielt.  
Sie werden:

1. Die **Spielumgebung** verstehen (`Game.py`),  
2. Den **Q-Learning-Agenten** kennenlernen (`QLearningAgent.py`),  
3. Den Agenten **trainieren** (im Notebook `Tic_Tac_Toe_RL_Agent.ipynb`) und  

---

## Schritt 0 – Setup

1. **Virtuelle Umgebung erstellen**  
```bash
python -m venv venv
```

2. **Aktivieren der Umgebung**  
- Windows: `venv\Scripts\activate`  
- Linux/macOS: `source venv/bin/activate`

3. **Abhängigkeiten installieren**  
```bash
pip install -r requirements.txt
```

---


**Hinweis:** Das Repository ist so aufgebaut, dass Sie Schritt für Schritt lernen, wie Reinforcement Learning funktioniert, ohne dass Sie direkt die komplizierten Details des Q-Learnings ändern müssen.


Studieren Sie nun das Tic_Tac_Toe_RL_Agent.ipynb File und versuchen Sie, die **Spielumgebung** (`Game.py`) zu verstehen. Werfen Sie anschliessend einen Blick in den **Q-Learning-Agenten** (`QLearningAgent.py`) und lernen Sie den kennen. Folgen Sie den Anweisungen aus dem Notebook `Tic_Tac_Toe_RL_Agent.ipynb`.
